Comparative stratification of causes of mortality during COVID-19 using CDC sources modulated by demographic reference from the Census Bureau. For reference on estimating mortality adjusted by demography see Ann Intern Med 2020 Dec 15. doi: 10.7326/M20-7385.

Methods

Calculations requiring the census API are drafted in the notebook listed above. The primary data sources are the same used for the parent mortality tracker application, suplemented by Counts of Deaths by Sex, Race, and Age. The reference data sources used in this analysis, with counts frozen and the end of July, are kept with open source in the public domain.
